# JEVis 4 - Images Folder

Dieses Verzeichnis enthält alle Bilder und Icons für die JEVis 4 Anwendung.

## Benötigte Dateien

### Favicon (Browser-Tab Icon)
Speichern Sie folgende Dateien in diesem Ordner:

- **favicon.ico** - Klassisches Favicon-Format (16x16 oder 32x32 Pixel)
- **favicon-16x16.png** - PNG Favicon für moderne Browser (16x16 Pixel)
- **favicon-32x32.png** - PNG Favicon für moderne Browser (32x32 Pixel)
- **apple-touch-icon.png** - Für iOS Home-Screen (180x180 Pixel)

### Logo (Sidebar und Header)
- **logo.png** - Hauptlogo der Anwendung (empfohlen: 32x32 Pixel, PNG mit transparentem Hintergrund)

## Bereits implementiert

Alle Icons sind bereits im HTML eingebunden:
- Favicon wird in allen Browsern angezeigt
- Logo erscheint in der Sidebar links oben
- Logo erscheint im mobilen Header
- Alt-Text ist konfiguriert: "JEVis Logo"

## Hinweise

- Verwenden Sie transparente PNG-Dateien für das beste Ergebnis
- Das Logo passt sich automatisch an Theme-Änderungen an
- Alle Dateien werden über den Pfad `/images/` geladen
